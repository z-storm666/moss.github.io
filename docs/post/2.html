超轻量级个人博客框架Gmeek

这是一个超轻量级个人博客模板。完全基于Github Pages 、 Github Issues 和 Github Actions。不需要本地部署，从搭建到写作，只需要18秒，2步搭建好博客，第3步就是写作。

不需要写代码，不用服务器，也不用域名，全流程都使用Github提供的免费服务。其中博客主体使用Github Pages托管的静态页面，构建部署使用Github Action，写文章则使用github的issue的富文本编辑器。
